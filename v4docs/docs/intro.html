Introduction

Background

Web Service Interface (WSI) allows external systems to read and update Click's data model using very simple SOAP messages. All Click stores have WSI enabled.

WSI is the lingua franca of the Click domain model.

WSI Message Format

The receiving response and the outgoing request have EXACTLY the same message format.

Incoming

<WebService Name="IRB">
  <Attribute Caption="Foo" Value="1234" />
</WebSrvice>

Outgoing

<WebService Name="CRMS">
  <Attribute Caption="Bar" Value="1234" />
</WebSrvice>
