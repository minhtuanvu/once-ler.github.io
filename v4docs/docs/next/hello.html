Hello World

There are plenty of online XSLT tutorials online like this one.

All examples that follow assume one has some familiarity with XSLT.

Namespace

All stylesheets require a namespace.

<?xml version="1.0" encoding="utf-8"?>
<xsl:stylesheet 
    version="1.0"
    xmlns:xsl="http://www.w3.org/1999/XSL/Transform"
    xmlns:x="http://www.w3.org/1999/xhtml">

    <!-- Your options go here -->

    <!-- Your code goes here. -->

</xsl:stylesheet>

Options

Always add these options.

<xsl:output 
    method="xml" 
    version="1.0"
    omit-xml-declaration="yes"
    encoding="UTF-8"
    indent="yes"/>
  
<xsl:strip-space elements="*"/>

WSI Root

For now, we'll just match the root ("/").

<xsl:template match="/">
    <xsl:element name="WebServiceRoot">
        <xsl:element name="WebService">
            <xsl:attribute name="Name">
                <xsl:text>IACUC_Protocol_IACUC</xsl:text>
            </xsl:attribute>
            <xsl:attribute name="TypeName">
                <xsl:text>_IACUC Study</xsl:text>
            </xsl:attribute>
        
            <!-- The attributes of the WSI goes here. -->
      </xsl:element>
    </xsl:element>
</xsl:template>

WSI Attributes

Finally, add the attributes to update.

<xsl:element name="Attribute">
    <xsl:attribute name="Caption">
        <xsl:text>Name</xsl:text>
    </xsl:attribute>
    <xsl:attribute name="DataType">
        <xsl:text>String</xsl:text>
    </xsl:attribute>
    <xsl:attribute name="Value">
        <xsl:text>Hello World!</xsl:text>
    </xsl:attribute>
</xsl:element>

Output

  <WebServiceRoot>
    <WebService>
      <Attribute 
          Caption="Name" 
          DataType="String" 
          Value="Hello World!"/>
    </WebService>
  </WebServiceRoot>
